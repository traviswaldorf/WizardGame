# Type System Overview

## Tier Hierarchy

The type system is organized into three tiers. Each tier builds on the previous, creating a web of elemental relationships grounded in physics and energy.

### Primary Types (Base Elements)

The four primaries map to classical states of matter and represent the foundational strategic archetypes.

| Type  | Old Name   | State of Matter | Strategic Archetype |
|-------|------------|-----------------|------------------|
| Fire  | Pyro       | Plasma          | Aggressive       |
| Earth | Kilo       | Solid           | Defensive        |
| Water | Flo/Hydro  | Liquid          | Flow             |
| Air   | Aero/Blo   | Gas             | Control          |

### Secondary Types (Combinations)

Each secondary type is formed by combining two primary types.

| Type     | Old Name | Composition   |
|----------|----------|---------------|
| Metal    | Forgo    | Fire + Earth  |
| Plant    | Gro      | Earth + Water |
| Ice      | Sno/Slo  | Air + Water   |
| Electric | Electro  | Fire + Air    |

### Tertiary Types (Dark/Energy Pairs)

Each of the 8 primary and secondary types has a tertiary "dark side" or energy counterpart. These represent the deeper physics domain each element connects to.

| Parent Type | Tertiary Type | Old Name        | Energy Domain         | Relationship        |
|-------------|---------------|-----------------|----------------------|---------------------|
| Fire        | Radioactive   | Radio           | Solar/Plasma          | Nuclear energy      |
| Earth       | Cosmic        | Cosmo           | Gravity/Solid         | Gravitational force |
| Water       | Poison        | Toxo            | Chemical/Liquid       | Chemical reactions  |
| Air         | Sound         | Stereo          | Pressure/Gas          | Acoustic waves      |
| Metal       | Crystal       | Slo/Tempo/Forgo | Structure/Bonds       | Lattice/resonance   |
| Plant       | Ghost         | Hollo/Shado     | Life/Organic          | Decay/death         |
| Ice         | Heat          | Thermo          | Thermodynamics        | Temperature         |
| Electric    | Magnetic      | Magneto         | Electromagnetism      | EM fields           |

## Augmenter Pairs (Outside Main System)

Two special pairs sit outside the 8-type hierarchy and act as augmenters/modifiers:

| Pair          | Role                                                                 |
|---------------|----------------------------------------------------------------------|
| Time - Space  | Temporal and spatial manipulation mechanics                          |
| Light - Dark  | Augmentations derived from the tertiary pairs (e.g., Poison = Dark Water) |

## Composition Rules

```
Fire  + Earth = Metal
Earth + Water = Plant
Air   + Water = Ice
Fire  + Air   = Electric
```

## Reaction Equations

```
Fire  + Poison      = Radioactive
Water + Radioactive = Poison
```

## Strategic Clusters

Types cross-cut into four thematic clusters that define playstyle tendencies:

| Cluster                       | Types               | Theme                        |
|-------------------------------|---------------------|------------------------------|
| Generation / Expand           | Ice, Water, Plant   | Resource growth, healing     |
| Control / Exploit             | Electric, Air, Ice  | Disruption, tempo control    |
| Protection / Explore          | Plant, Earth, Metal | Defense, tower building      |
| Aggressive / Exterminate      | Electric, Fire, Metal | Damage, destruction        |

Note: Some types appear in multiple clusters (Ice, Electric, Plant, Metal), creating strategic flexibility.
