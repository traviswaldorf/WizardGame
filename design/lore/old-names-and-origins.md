# Old Names and Origins

Each type has a lore-based "Old Name" using a thematic -o postfix convention. These names represent the practitioner title, class origin, or elemental identity.

## Primary Types

| Type  | Old Name   | Etymology                          |
|-------|------------|-------------------------------------|
| Fire  | Pyro / Igno       | Greek "pyr" (fire) / Latin "ignis" (fire)  |
| Earth | Kilo / Grano      | From kilogram — mass / Latin "granum" (grain) |
| Water | Flo/Hydro  | "Flow" / Greek "hydor" (water)      |
| Air   | Aero/Blo   | Greek "aer" (air) / "blow"          |

## Secondary Types

| Type     | Old Name | Etymology                               |
|----------|----------|-----------------------------------------|
| Metal    | Forgo / Ferro          | From "forge" / Latin "ferrum" (iron)       |
| Plant    | Gro / Vito / Bio       | From "grow" / Latin "vita" / Greek "bios"  |
| Ice      | Slo                    | From "slow" (tempo reduction)              |
| Electric | Electro / Volto / Ohmo | Greek "elektron" / "volt" (Volta) / "ohm" (Ohm) |

## Tertiary Types

| Type        | Old Name        | Etymology                              |
|-------------|-----------------|----------------------------------------|
| Radioactive | Fissio / Radio           | From "fission" / "radioactive"                    |
| Crystal     | Tempo                    | From "tempo" — controlling pace                   |
| Cosmic      | Cosmo / Astro / Nebulo   | Greek "kosmos" / Greek "astron" / Latin "nebula"  |
| Ghost       | Necro / Morto / Entro    | Greek "nekros" / Latin "mortis" / "entropy"       |
| Poison      | Toxo / Sporo             | Greek "toxikon" / Greek "spora" (spore)           |
| Heat        | Thermo / Calo            | Greek "therme" / Latin "calor" (heat)             |
| Sound       | Sono / Stereo            | Latin "sonus" / "stereophonic"                    |
| Magnetic    | Magneto / Polo           | From "magnet" / "polar/poles"                     |

## Naming Conventions
- The -o postfix is the unifying convention across all old names
- Some types have multiple candidate names (Water: Flo/Hydro, Air: Aero/Blo)
- Crystal's old name (Tempo) reflects its time/pace identity, distinct from Metal (Forgo)
