## Notes

- If the direct image URLs return 404s, the wiki may have reorganized. Visit the Type overview page and inspect the icon images to get current URLs.
- The image hash paths (e.g., `/images/6/6f/`) are based on MediaWiki's file hashing and may differ from what is listed above. The wiki pages themselves are the most reliable way to find current URLs.
- Save files as lowercase: neutral.png, wind.png, earth.png, water.png, fire.png, nature.png, electric.png, mental.png, digital.png, melee.png, crystal.png, toxic.png
