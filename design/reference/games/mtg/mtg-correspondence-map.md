# MTG Correspondence Map

Mapping Magic: The Gathering color/mechanic concepts to the Wizard Game element system. Organized by Wizard Game element to show what each type draws from.

## By Wizard Game Element

| MTG Mechanic                     | Wizard Translation          |
|----------------------------------|-----------------------------|
| Combat                           | Attack                      |

| Firebreathing (+X/+0 effects)    | Attack Boost                |
| Haste and Sacrifice at EOT       | Priority                    |
| Land Destruction                 | Element Source Destruction   |
| Mana Production                  | Short Term Element Boost    |
| Randomness                       | Coin flips for effects      |

## Types Without Clear MTG Mapping
These Wizard Game types are largely original design space:

- **Heat** — Thermodynamic acceleration has no MTG color equivalent
- **Sound** — "Players cannot speak" is entirely unique
- **Cosmic** — Gravity/space mechanics are original
