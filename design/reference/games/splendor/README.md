## System Overview

### Resource System
5 gem types + gold (wild):
| Gem | Color |
|-----|-------|
| Diamond | White |
| Sapphire | Blue |
| Emerald | Green |
| Ruby | Red |
| Onyx | Black |
| Gold | Wild (any color) |

### Core Loop
Take gems OR reserve a card (get gold) OR purchase a card → cards give permanent gem bonuses → attract nobles → score points

## Type / Element Interactions

Splendor has no type advantages — all gems are functionally equal. The interaction comes from **scarcity competition**: taking gems denies them to opponents.

## Competitive / Meta Insights

- **Engine efficiency is everything** — the best players build discount engines that make expensive cards free
- **Denying opponents** by taking gems they need is a valid strategy
- **Gold (wild) is the most powerful resource** because of flexibility
- **Noble strategy** (building toward specific noble requirements) provides direction to engine building
- **Speed vs quality trade-off** — rush cheap cards for fast nobles vs build a deep engine for tier 3 cards
