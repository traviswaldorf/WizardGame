## Best Source: Bulbapedia / Archives

The Bulbapedia archives host official-quality type icons. Visit each page and save the image.

**Type icon overview**: https://bulbapedia.bulbagarden.net/wiki/Type

## Alternative Sources

- **PokemonDB**: https://pokemondb.net/type — has type icons in the page
- **Serebii**: https://www.serebii.net/games/type.shtml — type chart with icons
- **PokeAPI sprites**: https://github.com/PokeAPI/sprites — community sprite repository

## Notes

- Wiki images often require a browser to download (user-agent restrictions on curl)
- Visit each source page, right-click the icon, and "Save image as" the indicated filename
- The SwSh (Sword/Shield) icons are the modern standard style
- Pokemon type icons are trademarks of Nintendo/Game Freak. Use for reference only.
